```ad-info
Der Prompt zielt darauf ab, ein Einführungskapitel für eine Notiz zu erstellen, das eine prägnante Zusammenfassung der Inhalte bietet und die inhaltliche Einordnung des Themas, den zentralen Gedanken, den Use Case, die Detailtiefe sowie die Zielgruppe klar darstellt. Er fordert eine strukturierte und informative Darstellung in maximal drei Absätzen, die den Leser direkt anspricht und unnötige Wiederholungen vermeidet.
```
---
```
Für die letzte Antwort:

ODER

"[Hier den zuvor eingegebenen Text einfügen]"
-----
Erstelle ein Einführungskapitel für diesen Text.
---
Achte dabei auf die folgenden Vorgaben:

Zweck:  
Liefere eine prägnante Zusammenfassung der Inhalte der Seite.

Inhaltliche Anforderungen:  
1. Beschreibe in maximal drei Absätzen die inhaltliche Einordnung des Themas.  
2. Erläutere den roten Faden der Seite: Was ist der zentrale Gedanke oder die Hauptaussage?  
3. Definiere den Use Case der Informationen: Warum sind diese Inhalte relevant, und wie können sie angewendet werden?  
4. Beschreibe die Detailtiefe der Inhalte: Sind die Informationen eher oberflächlich, praxisorientiert oder tiefgehend?  
5. Gib die Zielgruppe an: Für wen sind die Inhalte gedacht (z. B. Anfänger, Fortgeschrittene, Experten)?

Stil:  
- Sei prägnant, aber informativ.  
- Verwende eine klare und strukturierte Sprache, die den Leser direkt anspricht.  
- Vermeide unnötige Wiederholungen oder generische Aussagen.  
```
