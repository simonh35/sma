```ad-info
Der Prompt dient dazu, einen gegebenen Text klar und übersichtlich in Markdown zu formatieren, wobei die Hauptüberschrift mit Header 1 (#) beginnt und Untergliederungen logisch durch Header 2 (##) und Header 3 (###) strukturiert werden. Ziel ist es, eine gut lesbare, logisch gegliederte und standardkonforme Markdown-Darstellung zu erzeugen.
```
---
# Prompt
```
Für die letzte Antwort:

ODER

"[Hier Text eingeben.]"
-----
Formatieren Sie den folgenden Text als Markdown, beginnend mit einer Hauptüberschrift (#) und logisch gegliederten Unterüberschriften (##, ###). Trennen Sie Absätze klar und verwenden Sie bei Bedarf Aufzählungen oder Codeblöcke, um die Lesbarkeit zu verbessern. Achten Sie auf eine übersichtliche und standardkonforme Markdown-Darstellung. 
---
Binden Sie alle Code-Elemente in Markdown-Codeblöcke ein. 
---
Nummerieren Sie die Überschriften nicht. 
---
Geben Sie die gesamte Antwort in einer einzigen Markdown-Umgebung aus.
```
