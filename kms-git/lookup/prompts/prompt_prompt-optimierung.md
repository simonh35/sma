```ad-info
Der Prompt dient dazu, einen bereitgestellten Prompt iterativ zu analysieren und zu optimieren, um eine klarere, präzisere und qualitativ hochwertigere Version zu erstellen. Ziel ist es, die Struktur, den Inhalt, das Format und die Hierarchie des Textes zu verbessern und die finale Version in einer Markdown-Codeumgebung bereitzustellen.
```
---
# Prompt
```
Für die letzte Antwort:

ODER

"[Hier den Prompt einfügen, dessen Funktion zusammengefasst werden soll.]"

-----

Agiere als Senior Expert für Prompt Engineering und optimiere den bereitgestellten Prompt iterativ, um die bestmögliche Version zur Erreichung der Intention des Prompts zu entwickeln. Folge dabei diesen Schritten:

1. Analysiere den bereitgestellten Prompt und identifiziere Schwächen sowie Verbesserungspotenziale.
2. Überarbeite den Prompt schrittweise, um ihn klarer und präziser zu gestalten, sodass die Ergebnisqualität hinsichtlich Struktur, Inhalt, Format und Hierarchie optimiert wird.
3. Dokumentiere bei jeder Iteration die vorgenommenen Änderungen und die zugrunde liegenden Gründe.
4. Stelle die Zwischenergebnisse nach jeder Iteration dar.
5. Fahre fort, bis die finale Version erreicht ist, ohne auf Benutzereingaben zu warten.
6. Präsentieren die abschließende Version in einer Markdown-Codeumgebung. ```
