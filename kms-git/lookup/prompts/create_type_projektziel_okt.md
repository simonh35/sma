Der Prompt soll dazu dienen, ein klar definiertes Projektziel und messbare Ergebnisse für ein Projekt unter Anwendung der OKR-Methode zu formulieren. Ziel ist es, eine strukturierte und inspirierende Zielsetzung zu schaffen, die sowohl technische als auch nicht-technische Aspekte berücksichtigt und die Relevanz des Projekts für das Team und die Organisation hervorhebt.

# Prompt
```
[PROJEKTZIEL] = Projektziel eintragen 

———-

**Prompt zur Erstellung eines Projektziels nach der OKR-Methode**  
**Rolle:** Agiere als Senior Expert für Projektmanagement und formuliere das Projektziel für das folgende Projekt basierend auf der OKR-Methode (Objectives and Key Results).

**Input:** 

[PROJEKTZIEL]

**Anforderungen:**  

# Objective  

- Formuliere ein klares, inspirierendes und motivierendes Hauptziel für das Projekt.
- Das Objective sollte die langfristige Vision und den gewünschten Endzustand des Projekts widerspiegeln.  

# Key Results

- Definiere zwischen 3 und 5 messbare Ergebnisse, die zur Erreichung des Objectives beitragen.
- Jedes Key Result sollte spezifisch, quantifizierbar und innerhalb eines festgelegten Zeitrahmens erreichbar sein.  

**Zusätzliche Hinweise:**  
- Berücksichtige sowohl technische als auch nicht-technische Aspekte des Projekts.
- Hebe die Relevanz des Ziels für das Team und die Organisation hervor.
- Achte auf eine logische und klare Strukturierung der Objectives und Key Results.  

**Output Format:**  

# Objective:
[Dein formuliertes Objective hier]“  
# Key Results:
1. „[Key Result 1]“  
2. „[Key Result 2]“  
3. „[Key Result 3]“  
4. „[Key Result 4]“  
5. „[Key Result 5]“  
```
